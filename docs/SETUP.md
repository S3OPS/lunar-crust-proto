# MIDDLE-EARTH ADVENTURE RPG — Godot Setup Guide

**Note:** This project has been migrated from Unity to Godot. For legacy Unity setup instructions, see the bottom of this document.

## Quick Setup for Godot

### Step 1: Install Godot
1. Go to https://godotengine.org/download
2. Download **Godot 4.3** or later (Standard version)
3. Extract the downloaded file
4. Run the `Godot_v4.x_[platform].exe` (or equivalent for your OS)
   - No installation required! Godot runs as a standalone executable

### Step 2: Open the Project
1. Launch Godot
2. Click **"Import"** button in the Project Manager
3. Click **"Browse"** and navigate to the repository folder
4. Select the `project.godot` file
5. Click **"Import & Edit"**

### Step 3: Run the Game
1. Once the editor opens, press **F5** or click the "Run Project" button (▶️)
2. If prompted, select `scenes/main.tscn` as the main scene
3. The game will start!

## That's It!
- No accounts required
- No additional downloads
- No complex setup scripts
- Just download Godot and run!

## Controls
- **WASD**: Move your character
- **Mouse**: Look around
- **Shift**: Sprint (drains stamina)
- **Space**: Jump
- **Left Mouse Button**: Attack
- **Right Mouse Button**: Special ability (AOE attack, costs 30 stamina)
- **I**: Toggle inventory (coming soon)
- **C**: Toggle character sheet (coming soon)
- **J**: Toggle quest journal (coming soon)
- **M**: Toggle map (coming soon)
- **ESC**: Toggle mouse capture / Pause menu

## Gameplay Guide
### Getting Started
1. Start in the tutorial area
2. Use WASD to move and mouse to look around
3. Follow the on-screen prompts to learn the basic controls
4. Explore the world and complete quests!

### Tips
- Sprint conservatively - stamina regenerates slowly
- Special attacks are powerful but cost stamina
- Level up by defeating enemies and completing quests
- Save your game frequently (manual save coming soon)
